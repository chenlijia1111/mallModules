# mallModules

>商城模块化开发项目，方便后端快速开发
>
>将商城中通用模块抽出进行复用

### 模块

#### commonMallModule 

> 通用模块，包含商品，普通订单，用户地址，优惠券，评价，购物车，售后等通用模块

#### fightGroupModule

> 拼团模块

#### spikeModule

> 秒杀模块
